= NeoPower library for Arduino =

This library allows a microcontroller board to control a nearly unlimited number of DC motors with only one GPIO pin.

For more information about this library and the necessary hardware, please refer to the NeoPower documentation at
https://bastelstube.rocci.net/projects/MBS39_NeoPower/NeoPower.html
